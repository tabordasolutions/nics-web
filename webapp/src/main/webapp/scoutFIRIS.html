The Fire Integrated Real-time Intelligence System (FIRIS) Pilot Program provides enhanced wildfire situational awareness for first responders.
The program provides the Initial and extended response Incident Commanders (IC) access to intelligence and information to support the complex decision requirements for dynamic wildfires.
The FIRIS Program also combines Intelligence, Surveillance, and Reconnaissance (ISR) capabilities with accurate, real-time wildland fire spread modeling.
